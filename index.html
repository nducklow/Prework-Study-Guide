Prework Study Guide

HTML

  • The head element can't be seen by the user and contains information for browsers and developers.
  • The body element contains everything that the user will see.
  • The main element contains the core information related to the current webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can assign colors with hex # or with keywords like gray or green.
  • In our .css file you have to make sure to link it to the html document for changes to appear.
  • ".""class name""" Is used in .css file to modify all areas with the assigned class.

Git

  • "git checkout ""branch name""" Changes branch you are working in.
  • "git checkout -b ""branch name""" Creates a new branch and switches to it.
  • "git pull origin ""branch directory (usually main)""" Pulls down selected branch changes to local directory.
  • "git push origin ""branch directory (usually current feature branch)""" Pushes changes from local directory to selected branch.
  • "git add -A" Stages all changes to be commited.
  • "git commit -m ""message here""" Commits changes and adds a message to the commit.
